# CMPSC132-FinalProject
A Python implementation of the board game Snakes and Ladders. Supports more than two players, but two is recommended.

The goal of this game is to reach cell 100 first. In order to do that, players roll dice and move the number of cells as defined by the dice roll. If they land on a ladder, they will move up by some amount, however, landing on a snake will bring them down. 

To play, run the python file in a large terminal (you will need it :)). Select the number of players and player names (keep these short or else the game will be massive). 

Then simply follow the instructions from there and play the game. 

Who will make it to the top first?
